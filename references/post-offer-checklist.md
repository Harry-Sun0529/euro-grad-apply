# 拿到 Offer 后做什么：从决策到落地

**用法**：用户问"拿到 offer 了下一步"、"怎么办签证"、"行前准备"、"到了之后"等问题时读这个文件。

> 政策、签证细节、Sperrkonto 金额、健康保险供应商等**变化频繁**，AI 引用本文档具体数字前应主动 web search 最新值，并提醒用户以官方为准。

---

## 阶段 1：Offer 决策（最容易被低估）

### 多 offer 对比维度
按用户的毕业目标排序（详见 career-goals.md），重点考量：
- **项目本身**：课程匹配 > 学校排名（参考目标行业雇主认可度）
- **总成本 ROI**：学费 + 生活费 + 机会成本 vs 毕业薪资预期
- **工签 / 长期路径**：求职窗口长短、是否好转 PR（详见 country-quick-compare.md）
- **生活适配**：城市规模、气候、语言、当地华人社群密度

**避免单一维度决策**：
- 不要单看 QS 排名
- 不要单看学费
- 不要单看"听说 XX 好"

### Conditional Offer（条件式录取）
常见条件：
- **语言成绩补交**：通常给 deadline 前补到 IELTS 6.5+
- **本科学位完成**：在读生需毕业证 + 最终成绩单
- **认证报告**（CSSD/CIMEA）补交
- **资金证明**

**关键**：条件未满足 → offer 自动失效。读懂条件 → 倒推自己的执行节奏。

### Waitlist 处理
- 不要放弃：每年录取季都会有人放弃 offer 让 waitlist 转正
- **同时确认保底校的 deposit**：等 waitlist 不能耽误下一步
- 主动联系学校确认你的 waitlist 排名（部分学校会告知）

### Deposit / Enrollment Fee
- 多数学校接受 offer 时要交押金，€500-3,000 不等
- **接受了 deposit 一般不退**——除非你的签证被拒
- 接受 deposit 前确保你**真的会去**

### 拒绝其他 offer
- 简短礼貌的拒绝邮件即可
- 越早拒绝越好——给 waitlist 上的人机会
- 部分学校的拒绝有专门 portal 操作

## 阶段 2：接受 offer + 拿正式录取信

### 各国学校的"acceptance"流程
- **荷兰 Studielink 系**：在 Studielink 上 accept enrolment + 学校 portal 上 accept offer
- **德国大学 portal**：通常学校 portal 上点 accept，部分需邮寄确认信
- **瑞典 UA.se 系**：在 UA.se 上 reply to offer，4 月底-5 月初
- **法国 EEF 系**：5 月 31 日前在 EEF 上确认 1 个最终录取
- **意大利 Universitaly**：完成 pre-enrollment 即可，签证流程跟上
- **瑞士 ETH/EPFL**：邮寄录取确认 + 缴费

### 拿到的关键文件
- **Letter of Admission / Letter of Acceptance**：签证申请必备
- 学号 / Matriculation Number：用于校园系统、宿舍申请
- 学校的"sponsor letter"（适用荷兰 MVV 流程）

## 阶段 3：签证申请

> 各国签证类型、所需材料、资金证明额度年年变化。AI 在引用具体数字时**应主动 web search 最新政策**，并附官网链接。

### 德国 学生签证（D 类）
- 申请通道：德国驻华使馆 / 总领事馆（北京、上海、广州、成都、沈阳）
- **核心材料**：录取信 + APS + 资金证明（Sperrkonto）+ 健康保险 + 语言成绩
- **Sperrkonto 限制账户**：2026 年要求 €11,904/年（€992/月）
  - 主流供应商：Fintiba、Expatrio（在中国可在线开）
  - 费用 €49-280 不等
  - **落地后激活**：需在 Bürgeramt 注册地址 14 天内拿到 Meldebescheinigung 才能解冻
- 健康保险：选 TK/AOK/Mawista/DAK 之一
- 处理周期：4-12 周

### 法国 学生签证（VLS-TS 长居签）
- **资金证明**：每月最低 €615 × 留学月数；或担保人证明
- **住房证明**：前 3 个月住房（学校宿舍 / hôte attestation）
- 处理通道：TLS Contact / VFS Global（外包）
- **OFII / VLS-TS 在线验证（落地后核心）**：
  - 入境后 **3 个月内** 必须在 [ANEF 平台](https://administration-etrangers-en-france.interieur.gouv.fr/) 在线验证 VLS-TS
  - 不验证 = 签证作废
- 验证后可申请 **CAF 房补（APL/ALS）**——可覆盖租金 30-50%

### 荷兰 MVV + VVR（学校代办）
- **关键：学生本人不直接联系 IND**——学校作为 recognized sponsor 代办
- **资金证明**：至少 **€13,000/年** + 符合荷兰标准的健康保险
- 处理周期：60 天（旺季 90 天）
- 入境后到市政厅注册 → 拿 BSN → 学校再办 VVR 居留卡

### 瑞典 学生居留许可
- 通过 [Migrationsverket 在线申请](https://www.migrationsverket.se/)
- 资金证明：约 SEK 9,000+/月
- 健康保险：超过 1 年自动进入瑞典公共医疗（拿 personnummer 后）
- 处理周期：3-6 月，**越早交越好**

### 意大利 学生签证
- 在 Universitaly 完成 pre-enrollment 后
- 通过意大利驻华使馆/总领事馆走签证
- **36 小时快签**（部分签证类型 2024 起的新政策）
- 资金证明：约 €6,000+/年
- 健康保险：私立或意大利国家医疗（注册后）

### 瑞士 D 签证
- 申请：瑞士驻华使馆，材料齐全后 8-12 周
- 资金证明：CHF 21,000+/年
- 配额制——非 EU 学生有名额限制但学位项目通常不限

### 通用提醒
- **签证申请越早越好**：5-6 月开始，避免 7-8 月旺季拥堵
- **录取信原件可能需要邮寄到使馆**——留足邮寄时间
- **体检/无犯罪证明**：部分国家要求（如澳洲），欧洲多数硕士不强制
- **被拒处理**：仔细看拒信理由——多数是材料缺失，补齐重新交即可

### 已婚 / 带家属随行（30+ 申请者常见诉求）

欧洲多数国家允许学生签证带配偶 / 未成年子女随行，但**资金证明 + 等待期 + 配偶语言要求**差异很大。具体规则**每年变动，必须 web search 目标国当年官方政策**——下面是方向性参考：

**对家属随行较友好的国家**
- **法国**：配偶/子女与学生同步申签证；配偶可全职工作；子女上公立学校免费
- **德国**：允许 Familienzusammenführung（家庭团聚签），但配偶通常需先证明 A1 德语；Sperrkonto 金额按"两人份"算（具体年额查官网）；配偶持家属许可可全年工作约 120 全天 / 240 半天
- **荷兰**：可带家属，但部分大学/IND 要求**学生先在荷兰居住一段时间（如 1 年）**才能为配偶申请家属许可；配偶要工作需另办工签（2-4 月处理）
- **比利时、奥地利、爱尔兰**：流程相对清晰但同样需额外资金证明

**门槛较高 / 不友好**
- **瑞士**：非 EU 学生家属许可极少配额
- **北欧多国**：允许但配偶就业市场难度大；丹麦近年家属政策收紧

**家属随行的真实成本**
- 财力门槛：单人资金证明 × 1.5-2 倍
- 住房：单人 studio 不够，要 1-2 居 → 月租翻倍
- 配偶职业断层：配偶通常**不能立刻找到对口工作**，需要语言 + 资历再建立
- 子女教育：公立学校多数免费但要学当地语言；国际学校 €10,000-20,000+/年

**给 Skill 的回应原则**
当用户提"我有配偶 / 孩子要带"时：
1. 先问目标国家（家属政策差异极大）
2. 主动 web search 当前年的官方家属签证规则（金额、配偶语言、等待期）
3. 提醒**经济成本翻倍**——单纯比学费/生活费的对比表不适用
4. 引导用户去目标国移民局官网原文核对

### 资金证明的合规来源（签证拒签的常见原因）

资金"够数"只是表面——更关键的是**资金来源能否被使馆/移民局接受**。各国规则差异极大，下面是**基于官方移民局公示**的方向性参考：

> ⚠ **必须二次查证**：本节列出的是写文档时（2026 年）从官方网站查到的规则。具体每年表述可能微调。AI 给出建议时**必须主动 web search 当前年的官方政策**（auswaertiges-amt.de / ind.nl / Campus France / migrationsverket.se 等原文），并引导用户去官网核对。

**德国**（来源：auswaertiges-amt.de FAQ "Finanzierung"）
- 接受：父母收入和财产证明 / Sperrkonto 限制账户 / §66-68 AufenthG 担保声明（担保人需在德国有资产）/ 德国银行年度更新 Bürgschaft（担保函）/ 公共奖学金（DAAD、CSC 等）
- 多种方式可组合
- Sperrkonto 是中国学生最常走的路径

**荷兰**（来源：IND "Income requirements study"）
- 接受：本人或父母名下银行流水
- 资金必须**未被冻结、未挪作他用**
- 父母作为 financier 需提供：**IND private support statement** + 护照复印件 + 近 3 个月银行流水
- 学校作为 sponsor 替学生递交申请，资金证明随附

**法国**（来源：Campus France / france-visas.gouv.fr）
- 接受：本人近 3 个月银行流水 / attestation de prise en charge（担保声明）+ 担保人税单 + 工资单 / 奖学金证明
- 领事馆要求**资金来源稳定**（数月稳定收入，不是突然出现的存款）
- ⚠ enrollment（在读证明）/ financial proof（资金证明）/ housing（住房证明）/ insurance（保险证明）**的覆盖期限必须一致**，否则被拒

**瑞典**（来源：Migrationsverket "Maintenance requirement"）
- ⚠ **关键差异**：瑞典**不接受私人个体作为 sponsor**——父母不能担保！
- 接受：本人银行资产（流水 ≤4 个月）/ 奖学金 / 学生贷款 / 公司 / 组织 / 高校 sponsor / 已就业的工资收入
- 资金必须在**学生本人名下**——父母可以"赠与"但要在落地前转入学生账户
- 若学校提供免费住宿/餐饮，月维持额度可减免

**其他国家**
- 丹麦、芬兰、挪威、瑞士、奥地利、比利时等都有各自规则，**AI 不要套用德国/荷兰模式**——必须 web search 该国移民局原文

### 给中国学生的实操提醒

- **提前 3-6 个月** 让资金到位——突然出现的大额存款会被怀疑（"为这次签证临时凑的"）
- **保留资金来源解释**：父母工资单、税单、房产证明等，以备使馆补材料询问
- **国内→境外汇款**：单次 $50,000/年额度上限，分次汇 + 留凭证
- **资金证明 vs 真实生活**：拿到签证后这笔钱可以正常用，不是"必须冻结"
- **拒签后处理**：多数因资金理由被拒的可以补材料再申，但要看清拒信具体条款

### 给 Skill 的回应原则
当用户问"我需要准备多少钱才能办签证 / 资金证明怎么开"时：
1. 先确认目标国家（各国规则差异极大，尤其瑞典 vs 其他）
2. **主动 web search 当前年的官方资金证明要求**——金额每年都在调整
3. 明确告诉用户："以下信息是 [年份] 从官方查到的方向性参考，**做最终准备前请到官方网站核对当年具体条款 + 联系该国驻华使馆确认**"
4. 区分"资金够数"和"资金来源合规"——后者是更容易被忽视的拒签原因
5. 涉及父母担保时，特别提醒瑞典 + 提醒其他国家需要担保人详细材料

## 阶段 4：行前准备（出发前 2-3 月）

### 住房（最早开始安排）
- **荷兰、爱尔兰、北欧、瑞士的住房极紧张**——offer 一确认就开始找
- **学校宿舍**：通常先到先得，开放时间 4-6 月，**第一时间申请**
- **私人租房**：
  - 主流平台：Kamernet（荷兰）、WG-Gesucht（德国）、SeLoger（法国）、Boligportal（丹麦）、Blocket Bostad（瑞典）
  - 警惕**预付款骗局**——任何要求落地前转账定金的"中介"都要谨慎
  - 落地后 1-2 周临时住 Airbnb / hostel 也是常见过渡
- **中文房源**：小红书、各国 CSSA、华人微信群——信息更新快但水分多

### 资金 + 银行
- **国内带的钱**：少量现金（€500-1,000 现金 + €2,000+ 在国际信用卡 / Visa 借记卡）
- **国际信用卡**：建议至少 2 张（一张主用、一张备用）
- **国际汇款**：Wise、Revolut 是低费率选择
- **国内卡的预留**：保留至少 1 张国内卡能用 Apple Pay / Google Pay，过渡期付款方便

### 健康保险
- 各国学生签证通常要求覆盖整个学期/学年
- 部分国家可在落地后切换到本地学生保险（更便宜）
- 出发前买短期国际旅行医疗保险（如安联）覆盖落地前 1-3 月

### 必备文件原件 + 扫描备份
**必带原件**：
- 护照（**有效期建议大于回国机票日 + 6 个月**）
- 签证页（贴在护照里）
- 录取信原件
- APS 证书原件（德国）
- 学位证 + 成绩单 + CSSD 认证报告（中英文公证）
- 出生公证、亲子关系公证（中英文）
- 健康保险证明
- 资金证明（Sperrkonto / 银行存单等）
- 国际驾照（如需）

**扫描备份**：
- 全部上传到云盘（Google Drive / iCloud / OneDrive）
- 留一份给国内家人
- 重要文件复印 2-3 份，分开放

### 物品准备
**该带**：
- 转换插头（欧标圆形两孔）
- 常用药品（中药、感冒药、过敏药——欧洲药店要医生开处方）
- 眼镜备用一副 + 验光单
- 部分国家可带的中国调料（注意海关）

**不必带**：
- 大量衣物（占箱位，本地有更适合气候的）
- 大量现金（用国际卡更安全）
- 易碎或大件电器（用本地买的）

## 阶段 5：落地后第一个月（核心实务）

### 顺序最关键的 5 件事（以德国为例，其他国家类似但有差异）

1. **找到住处**（即使是临时的）
2. **市政厅注册地址**（Anmeldung 德国 / GBA 注册荷兰 / Folkbokföring 瑞典）
   - 这是后续一切（银行、保险、居留卡）的前置
   - 通常 14 天内必须完成
3. **拿 Meldebescheinigung / BSN / Personnummer 等"身份号"**
4. **激活银行账户**（Sperrkonto 解冻、本地账户开通）
5. **居留许可申请**（VLS-TS 验证 / VVR 拿卡 / 居留卡预约）

### 学校事务
- **入学注册**（Immatrikulation 德国 / Inschrijving 荷兰 / Inskrivning 瑞典）
- 拿学生卡 + 学期票
- 选课 + 加入新生群
- 部分学校有 orientation week

### 生活基础
- **电话卡 / SIM**：建议本地预付卡（如德国 Vodafone CallYa、荷兰 Lebara、瑞典 Comviq）；不必绑定长期合同
- **交通卡 / 学期票**：很多学校学费已含
- **超市 / 日常物品**：第一周适应物价
- **银行卡到货**：可能需 1-3 周邮寄

### 社交 / 心理适应
- **加入新生社团**（学生会 / 国际学生协会）
- **加入中国学生学者联合会（CSSA）**：信息互通 + 应急网络
- **不要 100% 依赖中国圈**：积极参加非中国背景的活动，建立多元社交
- **接受前 1-3 月不舒服**：文化冲击 + 行政繁琐 + 孤独感叠加是普遍现象，**不是失败**（详见 career-goals.md "心理预期管理"）

## 阶段 6：长期规划起点

### 第 1 学期开始时就思考的事
- **实习 / Werkstudent**：开学就投——欧洲找实习的窗口期短，开学前 1-2 个月就开始投
- **当地语言学习**：开课就报名（学校通常有免费/低价课程）
- **建立社交网**：导师、同学、当地朋友、华人朋友——分层经营
- **了解工签 / 长期路径**：详见 career-goals.md 和 permanent-residence-and-citizenship.md

### 第 1 学期完后的检查点
- 你是否还想留欧？/ 计划是否需要调整？
- 学业是否跟得上？需要不需要找学姐学长帮忙？
- 经济状况：是否需要找 Werkstudent / 兼职？
- 心理状况：是否需要找学校心理咨询服务？

## 阶段 5.5：金融 / 保险 / 银行 / 税务供应商对比

> ⚠ **重要免责声明**：以下供应商对比是 **方向性参考**——价格、产品功能、政策、销售活动、可用国家**每年都在变化**。这些选择直接关系到个人财务和健康，**做最终决定前必须**：
> - 到供应商官网核对当前条款 + 月费 + 费率
> - 在 Trustpilot / Reddit r/germany 等真实用户评测平台二次验证
> - 不要赶时间快速决定——多数选择有 7-14 天冷静期
> - AI 给出的具体金额和功能可能已过时——以官网为准

### 5.5.1 限制账户 / Sperrkonto（仅德国学生签需要）

主流两家供应商（Coracle 2025 起暂停新申请）：

| 供应商 | 设立费 | 月费 | 银行伙伴 | 特点 |
|---|---|---|---|---|
| **Fintiba** | ~€159 | ~€9.90/月 | Sutor Bank（德国，存款保险全覆盖）| 品牌信任度高、英文支持好 |
| **Expatrio** | ~€89 | ~€5/月 | 意大利伙伴银行 | 总价低 + 含套餐（含初期保险） |

- 两家都接受 €11,904 押金且被所有德国使馆认可
- **激活条件**：落地后到 Bürgeramt 注册地址（14 天内），拿 Meldebescheinigung 才能解冻
- **如何选**：预算敏感 → Expatrio；品牌优先 → Fintiba。两家整体都可用，选错不会出大问题，**关键是别拖到使馆约会前才办**

### 5.5.2 数字银行账户（德/法/荷/意/西等大部分欧洲国家可用）

落地后**强烈建议申请 1-2 个数字银行账户**，处理速度快、英文支持好、跨欧洲流动方便。

| 银行 | 类型 | 适合场景 | 关键限制 |
|---|---|---|---|
| **N26** | 持牌银行（BaFin 监管，存款保险 €100K）| 主账户用，德/奥/法/荷/意/西居民友好；免费版有德国 IBAN | 免费版每月 2 次 ATM 免费，超出 €2/次；EU 外汇兑换有 markup |
| **Revolut** | 持牌银行（立陶宛）| 多币种 / 旅行 / 投资交易；免费版含 25+ 货币兑换 | 部分场景被欧洲商户当"非传统银行"对待；客服质量参差 |
| **Bunq** | 持牌银行（荷兰）| 多 IBAN / 可持续投资偏好者 / expat | **无免费版**（最低 ~€3-10/月）|
| **Wise** | 电子货币机构（**非银行存款保险**）| 国际汇款、freelancer 多币种收款 | 不受 DGSD 保护 → **不要存大额**；汇率低但不能当主账户 |

**常见组合**：
- "N26（主账户）+ Wise（国际汇款）"——大多数学生够用
- "N26 + Revolut"——常旅行者额外加 Revolut
- 不要把 Wise 当作长期存款账户

**德国本地传统银行**（Deutsche Bank、Sparkasse、Commerzbank）：办卡慢、英文支持差、月费高，**不推荐学生**。除非雇主 / 房东特别要求。

### 5.5.3 健康保险

#### 德国（核心选择：公保 GKV vs 私保 PKV）

**公保（GKV）—— 推荐 30 岁以下、有学生身份的人**
- 法定核心保障在所有公保几乎一致（联邦法律规定）
- 月费约 €110-150（2026 参考）
- 主要差异在：英文支持、APP 体验、附加福利、奖金计划
- 主流四家：
  - **TK**（最受国际学生欢迎）：英文客服 + APP 体验最好，11M+ 会员
  - **AOK Bayern**：分支网络最广（线下办事方便）
  - **DAK**：奖金计划最慷慨（运动健身补贴可达 €500/年）
  - **Barmer**：24/7 远程视频问诊 + 心理健康 APP
- ⚠ **30 岁后公保学生费率优惠失效**——切换 PKV 或正常 GKV 价格（月费翻倍）

**私保（PKV）—— 适合 30+ 学生、Sprachkurs 阶段、短期学位**
- 主流：MAWISTA Student（~€28+/月）、Care Concept（~€26+/月，最便宜）
- ⚠ **重要限制**：多数学生 PKV **不覆盖心理治疗 / 长期慢性病**——如果有这些需求，看 HanseMerkur / Provisit
- PKV 上学期有效后**不能轻易切回 GKV**——长期定居者要慎选

#### 荷兰

- **学生 + 无工作 / 无实习** → 不能买荷兰公保（basisverzekering）
- 必须买私立国际学生保险，推荐：**AON Student Insurance**（€40-60/月，多数大学合作伙伴）、IPS、Allianz Care
- **一旦有兼职 / 实习** → 必须切换到荷兰公保（月费 €148-159 + €385/年自付额）
- 切换需在工作开始 4 个月内完成，否则补缴罚款

#### 法国

- **学生默认免费加入法国 CPAM 公保**（不分国籍）→ 报销 70% 医生费用 + 部分药费
- 但 CPAM 注册要 1-3 月才生效——期间用过渡保险
- **强烈建议加 mutuelle（互助补充保险）**：MGEN、LMDE、SMENO、SMERRA 等
- mutuelle 月费 €15-50，覆盖 CPAM 报销之外的 30%
- 不买 mutuelle 也不会拒诊，但牙科 / 眼科自费会很贵

#### 其他国家
- **瑞典 / 北欧**：长期居留（>1 年）自动进入公共医疗（拿 personnummer 后），落地前用国际旅行保险过渡
- **意大利**：可选加入 SSN 国家医疗系统（€700+/年）或买私保
- **瑞士**：必须买私立健康保险（CHF 200-400/月，是欧洲最贵），落地后 3 个月内必办
- **国别细节年年变** → 必须 web search 当年规则

### 5.5.4 国际汇款（从国内到欧洲 / 反向）

| 工具 | 费率参考（€1,000）| 速度 | 备注 |
|---|---|---|---|
| **Wise** | ~0.4-0.6% + 固定费 | 几小时-2 天 | 透明 + 中端最便宜，强烈推荐 |
| **Revolut** | 免费版每月 €1,000 免费换汇额度 | 即时 | 适合小额、常用 |
| **国内银行电汇** | ~1-3% + €30-50 | 1-3 工作日 | 慢 + 贵，但接受度高 |
| **Western Union** | 高 + 不透明 | 即时 | 紧急时用，**不推荐常规** |
| **支付宝 / 国际版** | 部分场景免费 | 即时 | 仅限部分国家、部分商户 |

**给中国学生的现实提示**：
- 国内银行单笔结汇限额 $50,000/年（注意年度额度）
- 跨境汇款超过 $50,000 等值人民币需走特殊申报
- **每月分次汇** 比一次大额合规度高
- 留存所有汇款凭证（学校 / 移民局可能要求资金来源证明）

### 5.5.5 税务（简短）

> ⚠ 留学生税务因国而异、因学位类型而异、因有无工作而异，**强烈建议第一年咨询当地税务师 / 学校 international office**。下面只是基础框架。

**普遍原则**
- 学生身份本身**没有税务申报义务**（只读书无收入）
- 一旦有 **兼职 / Werkstudent / 实习收入** → 进入税务体系
- 多数欧洲国家有**学生工时上限**（如德国 20 小时/周学期内、120 全天/年）
- 中国 vs 多数欧洲国家有 **避免双重征税协定**——同一笔收入只在一边交税

**典型情况**
- **德国**：学生工资月不超 ~€538（Minijob 上限 2026 参考）→ 免税；超出 → 按累进税率
- **荷兰**：30% ruling 仅对**毕业后正式工作**适用，学生兼职不适用
- **法国**：留学生兼职月不超过 SMIC 一定比例 → 免税额度内
- **瑞典 / 北欧**：学生兼职超过年免税额（~SEK 24,000）需申报

**重要提醒**
- 拿到工资后**主动**问雇主你的税号 / 社保号是否注册
- 不要把"零收入"等于"不用申报"——部分国家要求**零收入也要申报**
- 跨境收入（如国内远程工作 + 欧洲生活）税务复杂，必须找税务师
- 入籍前后税务规则可能改变，**长期定居者需重新评估**
