# 欧洲留学申请时间线总览

---

## 秋季入学（9-10月入学，最常见）

### 提前 18-12 个月（大三下 / 毕业前一年的春季学期）

**核心任务：明确方向 + 语言准备 + 初步选校**

- [ ] 确定留学目标：国家、学位层次（硕/博）、学科方向
- [ ] 评估自身条件：GPA、语言水平、研究/实习经历、预算
- [ ] 开始语言考试准备：IELTS/TOEFL（建议目标 7.0/100+）
- [ ] 如需小语种（德语/法语等）：更早开始学习
- [ ] 初步选校：列出 10-15 个目标项目，查询具体要求和截止日期
- [ ] 如申请德国：开始准备 APS 材料（公证、翻译等）——注意 2024 年起可选 TestAS 代替面审
- [ ] 如申请博士：开始阅读文献、明确研究方向
- [ ] 研究奖学金信息，制作奖学金日历

### 提前 12-9 个月（大四上学期 9-12月）

**核心任务：完成语言考试 + 启动申请 + 准备材料**

- [ ] 完成语言考试（建议 10-11月前出分，为签证和申请留余量）
- [ ] 如申请德国：完成 APS 审核（面审或 TestAS，建议 9-10 月完成，VPD 办理约需 4 周）
- [ ] 准备申请材料：
  - [ ] CV 定稿
  - [ ] Motivation Letter 初稿（先写一个通用版本，再针对每个项目定制）
  - [ ] 联系推荐人（提前 2-3 个月），提供你的 CV 和申请项目列表
  - [ ] 准备成绩单英文版、在读证明
- [ ] 如申请法国：注册 Campus France / Études en France 系统
- [ ] 瑞典：universityadmissions.se **10月中旬** 开放，准备材料
- [ ] 荷兰：部分项目 **10月** 即可提交
- [ ] 博士方向：开始套磁、搜索 PhD vacancy、撰写 Research Proposal
- [ ] 如申请设计/艺术：开始整理和完善作品集

### 提前 9-5 个月（12月-次年4月）

**核心任务：提交申请 + 奖学金申请**

- [ ] **12月中旬**：ETH Zürich、EPFL 部分项目截止
- [ ] **12月-1月**：提交德国大部分申请（通过 Uni-assist 或学校 portal）
- [ ] **1月15日**：瑞典 universityadmissions.se 统一截止
- [ ] **1月-2月**：
  - [ ] 法国 Grande École 和部分项目截止
  - [ ] Erasmus Mundus 项目多在此期间截止
  - [ ] SI 奖学金（瑞典）截止
  - [ ] Holland Scholarship 截止
- [ ] **2月-3月**：
  - [ ] 荷兰很多项目截止（有些到4月1日或5月1日）
  - [ ] 比利时 KU Leuven 等校截止
- [ ] **3月**：CSC 国家公派申请提交（需已有外方录取/邀请信）
- [ ] 补交材料、Track 申请状态
- [ ] 各奖学金申请（截止日期各不相同，需逐一查看）

### 提前 5-2 个月（4-7月）

**核心任务：确认录取 + 签证 + 住房**

- [ ] **4月初**：瑞典统一出结果
- [ ] **4-5月**：陆续收到其他学校录取通知
- [ ] 确认 offer（注意各校的 acceptance deadline，有些需要缴纳 deposit）
- [ ] **5-6月**：CSC 公布结果
- [ ] 开始办理签证：
  - [ ] 准备资金证明
  - [ ] 德国：开设 Sperrkonto（限制账户，2024年9月起 €11,904/年即 €992/月）
  - [ ] 法国：完成 Campus France 签证流程
  - [ ] 荷兰：学校代办 MVV
  - [ ] 购买/确认健康保险
- [ ] 安排住房（越早越好！荷兰、瑞典住房紧张）
- [ ] 如有奖学金录取，完成相关手续

### 提前 2-0 个月（7-9月）

**核心任务：签证 + 行前准备**

- [ ] 签证面签（如需要，通常 7-8月）
- [ ] 机票预订
- [ ] 行李准备：
  - [ ] 重要文件原件（成绩单、学位证、录取通知、APS 证书等）
  - [ ] 少量现金 + 国际信用卡/借记卡
  - [ ] 转换插头（欧标圆形两孔）
- [ ] 在线完成入学注册（Enrollment/Immatrikulation）
- [ ] 加入新生微信群（通过 CSSA 或小红书找到）
- [ ] 到达后：
  - [ ] 报到注册
  - [ ] 办理居留许可/延长签证
  - [ ] 开银行账户
  - [ ] 办理电话卡
  - [ ] 购买交通卡/学期票
  - [ ] 熟悉城市和学校

## 春季入学（2-4月入学，部分国家/项目可选）

- **德国夏季学期**（4月入学）：申请截止通常在前一年 **11月-次年1月**
- **部分荷兰项目**有 2月入学
- **北欧极少有春季入学选项**
- 选择春季入学的理由：秋季没赶上申请、gap semester 安排、特定项目只有春季入学

---

## 特殊时间节点提醒

### APS 审核（德国申请者必看）
- 预约到面审可能需等 **1-3 个月**
- 出结果再等 **4-8 周**
- 总共可能需要 **4-6 个月**
- 建议大三下学期就开始准备材料

### CSC 国家公派
- 通常 **3月** 截止网申
- 需提前拿到外方录取/邀请信
- **5-6月** 公布结果
- 意味着你需要在前一年秋季就开始联系外方导师/学校

### Erasmus Mundus
- 各项目截止日期不同，多在 **1月-2月**
- 需要为每个申请项目单独准备材料
- 部分项目有多轮录取

### 作品集类项目
- 可能有额外的 home assignment（如 Design Academy Eindhoven）
- 部分有入学考试/面试（如法国/瑞士部分院校）
- 需要预留 **1-2个月** 的额外准备时间

### 博士申请
- 很多 PhD vacancy 是 **不定期发布** 的（不像硕士有统一申请季）
- 需要持续关注岗位平台和导师主页
- 套磁可以在任何时候开始，但建议在你准备好 CV 和初步研究想法后

## 多国同时申请的时间管理

如果你同时申请多个国家，以下策略有助于管理时间：

1. **先列出所有截止日期**，按时间排序
2. **Motivation Letter 策略**：先写一个通用框架，再针对每个项目定制 program-specific 段落
3. **推荐信**：提前统一联系推荐人，给出所有项目的截止日期列表
4. **成绩单/证件**：一次性多开几份，因为不同系统可能需要不同格式
5. **使用表格或 Notion 等工具追踪每个申请的状态**：
   - 项目名、截止日期、所需材料、提交状态、结果

## 关键月份速查

| 月份 | 关键事项 |
|------|----------|
| 9月 | Polimi 首轮开放；德国 APS 面审（建议时间） |
| 10月 | 瑞典 universityadmissions.se 开放；荷兰部分项目开放 |
| 11月 | DAAD 奖学金截止；瑞士政府奖学金截止 |
| 12月 | ETH/EPFL 截止；德国部分项目截止 |
| 1月 | 瑞典统一截止(15日)；Eiffel 提名截止；EMJMD 截止 |
| 2月 | Holland Scholarship 截止；SI 奖学金截止 |
| 3月 | 荷兰/比利时部分项目截止；CSC 网申截止 |
| 4月 | 瑞典出结果；陆续收到 offer |
| 5月 | 确认 offer；CSC 结果 |
| 6-7月 | 签证申请；住房安排 |
| 8-9月 | 行前准备；出发 |

## KAUST 时间线补充

- KAUST 不按欧洲统一申请季组织，必须查官方 Admission Timeline。
- 2026-06-18 查询官网时：2026 entry 已关闭，2027 entry 预计 8 月中开放。
- 中国籍申请人需结合 2026 季中文申请端“录取体感收紧”的反馈提前准备，不要等欧洲申请结束后再临时补 KAUST。
