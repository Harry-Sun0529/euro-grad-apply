# 申请文书写作指南

---

## 1. Curriculum Vitae (CV)

### 格式
- 1-2 页（硕士申请 1 页足够，博士或经验丰富者可 2 页）
- 清晰简洁，使用 bullet points
- 德国/奥地利部分项目传统上需附照片，其他国家不需要
- 推荐使用 LaTeX 模板（Overleaf 上搜 "Academic CV"）或简洁的 Word 模板
- 博士申请推荐使用 **Europass CV 模板**（europass.eu），欧盟推荐格式
- 字体统一，排版整洁，PDF 格式提交

### 结构建议
1. **个人信息**：姓名、联系方式（邮箱+电话）、国籍、LinkedIn（如有）
2. **教育背景**（倒序）：学校名、专业、学位、时间段、GPA（标明满分制）、排名（如有）、核心课程（可选）
3. **研究经历**：项目名称、指导教师、时间段、你的具体工作、方法/工具、成果
4. **实习/工作经历**：公司/机构名、职位、时间段、主要职责和成果
5. **发表/学术成果**（如有）：论文引用格式，标明你的作者位置
6. **技能**：语言（标注水平如 IELTS 7.0）、编程语言、软件工具
7. **获奖与荣誉**：奖学金、竞赛获奖等
8. **课外活动**（如有空间且相关）

### 中国学生常见问题
- **学生会经历写太多**——欧洲学术申请更看重学术/专业经历，学生会干部经验可以提但不要占主体
- **GPA 未注明满分制**——务必标明如 "88/100" 或 "3.6/4.0 (Top 10%)"
- **研究经历描述太笼统**——不要写"参与了XX项目"，要写具体做了什么：使用什么方法、处理什么数据、得到什么结论
- **格式不统一**——日期格式、bullet 格式、字体大小保持一致
- **信息过多**——不要写高中及以前的经历（除非特别突出的竞赛获奖）

## 2. Motivation Letter（动机信）/ Statement of Purpose

### 核心框架：回答三个问题
1. **Why this field?** 你为什么对这个学科/方向感兴趣？什么经历激发了你？
2. **Why this program/school?** 这个项目有什么具体吸引你的地方？课程设置、教授、实验室、地理位置/行业集群？
3. **Why you?** 你有什么背景和能力让你适合这个项目？毕业后打算做什么？

### 结构（约 500-1000 词，1-2 页）

**开头（1段）**
- 引入你的学术兴趣/背景
- 可以用一个具体的经历或问题作为 hook
- 但不要太文学化——欧洲学术申请偏好直接清晰

**中段 1：你的背景（1-2段）**
- 你的学术/研究/实践背景，如何为申请这个项目做了准备
- 具体的项目、课程、实习经历——不是重复 CV，而是讲 CV 背后的"why"和"how"
- 展示你在这个领域的思考和成长

**中段 2：为什么选这个项目（1段）**
- 展示你做了功课：提到 **具体的课程名、教授名、研究组、项目特色**
- 解释这些要素如何与你的兴趣和目标匹配
- 不要写"贵校排名高"——这是最无效的理由

**结尾（1段）**
- 你的职业/学术规划
- 这个项目如何帮助你实现目标
- 简洁有力，不要拖沓

### 各国风格差异
- **德国**：偏务实、逻辑清晰，不需要太多"故事性"，注重课程匹配和学术准备。德国人喜欢条理分明的论述
- **荷兰**：结构化、清晰、直接，可以有一些个人特色但不要过度
- **法国**（尤其 Grande École）：可以更 personal，讲述个人成长叙事，展示文化修养和全球视野
- **北欧**：简洁、真诚、不要过度包装。瑞典 SI 奖学金文书需突出 leadership 和 sustainability
- **瑞士**（ETH/EPFL）：非常学术导向，重点讲学术准备和研究兴趣

### 中国学生常见问题
- **过度谦虚或过度自夸**——找到平衡：自信 + 有据可依
- **开头写"从小我就对XX充满热情"**——太 cliché，所有人都这么写
- **只写自己多优秀但不解释为什么选这个项目**——要展示 match，不是单方面自我推销
- **把 CV 上的内容重新叙述一遍**——动机信应补充 CV 展示不了的"为什么"和"怎么想的"
- **中介代写导致千篇一律**——务必自己写，展示真实思考。审阅者见过无数模板化的信
- **每个项目用同一封信**——至少 program-specific 段落必须定制，提到该项目独有的东西
- **语法错误过多**——请 native speaker 或英语好的朋友/老师帮忙校对

## 3. 推荐信

### 选择推荐人
- **学术推荐人优先**：上过你的课 + 指导过你的项目/论文的教授
- 理想组合：2 封学术 + 1 封实习/工作（如适用）
- 推荐人的 **知名度不如推荐信内容重要**——一封写得很具体、很了解你的信 > 一封大牛的泛泛之词
- 选择真正了解你的教授，而非职位最高但与你接触少的教授

### 沟通策略
- **提前 2-3 个月** 联系推荐人
- 给推荐人提供：
  - 你的 CV
  - 成绩单
  - 申请项目列表和截止日期
  - 你希望他/她强调的方面（如研究能力、独立性、创造力等）
  - 简要描述你在他/她课上/项目中的表现
- 定期提醒截止日期（礼貌地）

### 中国学生特殊情况
- 部分教授可能让你 **自己起草** 推荐信——这在中国比较常见
- 如果是这样：
  - 写具体的学术互动和你的能力（不要写空洞的赞美）
  - 以教授的视角写，引用具体的课程/项目经历
  - 让教授在此基础上修改和签名
  - 不同推荐信的内容和风格要有差异化
- 有些欧洲项目需要推荐人 **直接上传到系统**——提前告知推荐人操作流程和截止日期
- 部分推荐人可能不熟悉在线提交系统——必要时帮助推荐人操作

### 推荐信内容要素
一封好的推荐信应包含：
- 推荐人与你的关系（上过什么课/指导过什么项目）
- 你在课程/项目中的具体表现（用例子说明，不是泛泛而谈）
- 你的突出能力（分析能力、创造力、独立思考、团队合作等）
- 与同水平学生的比较（如"top 5% of students I have taught"）
- 对你申请该项目的推荐意见

## 4. Research Proposal（博士申请）

### 结构（通常 2000-5000 词，因项目而异）

1. **Title**：简洁、准确、信息量大
2. **Introduction / Background**（约 500-1000 词）：
   - 研究领域概述
   - 现有文献综述（展示你对领域的了解）
   - 研究空白/问题（gap）
3. **Research Questions**（约 200-300 词）：
   - 明确的研究问题（1-3 个）
   - 每个问题为什么重要
4. **Methodology**（约 500-1000 词）：
   - 你计划如何回答这些问题
   - 研究方法/实验设计/数据来源
   - 为什么选择这种方法
5. **Expected Contributions**（约 200-300 词）：
   - 预期的学术贡献/创新点
   - 潜在的实际应用（如适用）
6. **Timeline**（如要求）：
   - 各阶段的时间安排（通常按年/季度划分）
7. **References**：
   - 参考文献列表（展示你读了多少相关文献）

### 关键原则
- 展示你对领域有 **深入了解**——引用近 3-5 年的关键文献
- 研究问题要 **specific enough to be doable** 但 **significant enough to matter**
- 方法论要可行，不要画太大的饼
- 如果申请特定导师的 position，研究计划要和 **导师的方向高度契合**
- 如果导师已定义了项目，你写的更像是你对该项目的理解和你能带来什么
- 语言要学术但清晰，避免不必要的行话堆砌

## 5. 作品集 / Portfolio（艺术、设计、建筑类）

### 基本要求
- 通常 **15-30 件作品**，PDF 格式，20-50 页
- 展示从概念到成品的 **完整过程**（process work 在欧洲非常重要）
- 排列顺序：最强的放前面和最后
- 每件作品附简要说明：项目背景、你的角色、使用的工具/材料、设计意图

### 注意事项
- **各校要求差异大**——务必仔细阅读项目的 portfolio guidelines
- Polimi 设计类可能要求在线提交特定格式或有额外测试
- 荷兰 Design Academy Eindhoven 通常还需要 **home assignment**（设计作业）
- 德国部分公立艺术学院要求 **Mappe**（实体作品集）寄送
- 瑞士/法国部分院校有 **入学考试（concours/Aufnahmeprüfung）**

### 欧洲作品集的偏好
- **展示思维过程** 比展示最终成品更重要
- 跨学科/跨媒介的实验精神在欧洲受欢迎
- 概念驱动（concept-driven）而非纯技术展示
- 社会关怀/可持续设计理念加分
- 清晰的叙事逻辑——作品集本身就是一个设计作品

### 格式建议
- PDF 文件大小控制在 10-20MB（除非有特殊要求）
- 页面尺寸 A4 或 Letter
- 封面：姓名、申请项目、联系方式
- 目录页（如有多个项目）
- 每个项目 3-6 页：概念→过程→成品
- 统一的视觉风格和版式

## 6. GPA 换算参考

### 百分制 → 4.0 制（常见算法之一）
| 百分制 | 4.0 制（参考）|
|--------|--------------|
| 90-100 | 4.0 |
| 85-89  | 3.7 |
| 82-84  | 3.3 |
| 78-81  | 3.0 |
| 75-77  | 2.7 |
| 72-74  | 2.3 |
| 68-71  | 2.0 |
| 64-67  | 1.5 |
| 60-63  | 1.0 |

注：不同学校的换算标准可能不同，以上仅供参考。建议在 CV 和成绩单中标明原始分数和满分制。

### 巴伐利亚公式（德国常用）
```
德国成绩 = 1 + 3 × (最高分 - 你的分数) / (最高分 - 及格线)
```
- 通常：最高分 = 100，及格线 = 60
- 德国 1.0 = 最好，4.0 = 及格
- 很多德国硕士项目要求换算后 2.5 以下（对应约 80/100）

### WES 认证
- 部分项目要求通过 WES（World Education Services）进行学历和成绩认证
- WES 会出具一份按美国/加拿大标准换算的成绩报告
- 费用约 $200-300，处理时间约 2-4 周
- 需要先做中国学位认证（通过 CSSD/学信网，CDGDC 已于 2022 年 8 月停办，CHESICC 已更名为 CSSD —— 详见 china-specific-procedures.md）
